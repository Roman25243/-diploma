# 🔐 Як увійти в адмін-панель

## Крок 1: Створити адміністратора

Запустіть скрипт створення адміна:

```powershell
cd d:\Кузь\cinema_app
& "d:\Кузь\.venv\Scripts\python.exe" create_admin.py
```

Або якщо віртуальне середовище активовано:

```powershell
python create_admin.py
```

## Крок 2: Введіть дані

Скрипт запитає:
- 📧 **Email адміна** (наприклад: admin@cinema.com)
- 👤 **Ім'я** (наприклад: Admin)
- 🔐 **Пароль** (мінімум 6 символів)

## Крок 3: Увійдіть в систему

1. Відкрийте браузер і перейдіть на http://localhost:5000
2. Натисніть **"Вхід"** у верхньому меню
3. Введіть email та пароль адміна
4. Після входу ви побачите кнопку **"Адмін"** у навігації

## Крок 4: Відкрийте адмін-панель

Адмін-панель доступна за адресою:
- 🔗 http://localhost:5000/app/admin

Або натисніть на кнопку **"Адмін"** (⚙️) у верхньому меню.

## 🔧 Альтернативний спосіб (через Python shell)

Якщо скрипт не працює, можна створити адміна вручну:

```python
# Запустіть Python shell
cd d:\Кузь\cinema_app
& "d:\Кузь\.venv\Scripts\python.exe"

# В Python shell виконайте:
from app import app
from extensions import db
from models import User
from werkzeug.security import generate_password_hash

with app.app_context():
    admin = User(
        email='admin@cinema.com',
        password=generate_password_hash('admin123'),
        name='Admin',
        is_admin=True
    )
    db.session.add(admin)
    db.session.commit()
    print("Адміна створено!")
```

Потім вийдіть з Python shell:
```python
exit()
```

## ❓ Поширені проблеми

### "Адміністратор вже існує"
Це означає, що адмін вже створений. Скрипт покаже email існуючого адміна.

### "Користувач з email вже існує"
Скрипт запропонує зробити існуючого користувача адміністратором.

### Забули пароль?
Потрібно вручну змінити пароль в базі даних або видалити користувача та створити нового.
