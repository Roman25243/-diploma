# ⭐ Система рейтингів та відгуків

## 📋 Що реалізовано:

### ✅ Функціонал:

1. **Рейтинг фільмів (1-5 зірок)**
   - Користувачі можуть оцінювати фільми від 1 до 5 зірок
   - Автоматичний підрахунок середнього рейтингу
   - Відображення кількості відгуків

2. **Текстові відгуки**
   - Можливість написати коментар до оцінки
   - Відображення дати та часу відгуку
   - Автор відгуку відображається

3. **Обмеження**
   - Один користувач = один відгук на фільм
   - Тільки авторизовані користувачі можуть залишати відгуки
   - Можна видалити свій відгук (або адмін може видалити будь-який)

4. **Відображення**
   - Рейтинг на картці фільму (список фільмів)
   - Детальний великий рейтинг на сторінці фільму
   - Список всіх відгуків на сторінці фільму (сортування: нові першими)

---

## 🗄️ Нова модель:

```python
class Review(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    user_id = db.Column(db.Integer, db.ForeignKey('user.id'), nullable=False)
    film_id = db.Column(db.Integer, db.ForeignKey('film.id'), nullable=False)
    rating = db.Column(db.Integer, nullable=False)  # 1-5
    comment = db.Column(db.Text)  # Необов'язково
    created_at = db.Column(db.DateTime, default=datetime.utcnow)
```

---

## 🎯 Як використовувати:

### Для користувачів:

1. **Переглянути рейтинг:**
   - Відкрити будь-який фільм
   - Рейтинг показується вгорі сторінки

2. **Залишити відгук:**
   - Авторизуватись
   - Відкрити сторінку фільму
   - Прокрутити до форми "Залишити відгук"
   - Вибрати оцінку (1-5 ⭐)
   - (Опціонально) Написати коментар
   - Натиснути "Опублікувати відгук"

3. **Видалити свій відгук:**
   - Знайти свій відгук в списку
   - Натиснути "Видалити"

### Для адміністраторів:

- Адміни можуть видаляти будь-які відгуки (модерація)

## 🚀 Маршрути:

- `GET /film/<id>` - перегляд фільму з відгуками
- `POST /film/<id>/review` - додати відгук
- `POST /review/<id>/delete` - видалити відгук

## 🔮 Можливі доповнення:
