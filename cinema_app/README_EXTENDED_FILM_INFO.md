# 🎬 Розширена інформація про фільми

## 📋 Що реалізовано:

### ✅ Нові поля моделі Film:

1. **genre** (Жанр)
   - Тип: String(100)
   - Приклад: "Бойовик", "Комедія", "Драма"
   - Відображається як бейдж на картках фільмів

2. **director** (Режисер)
   - Тип: String(100)
   - Приклад: "Крістофер Нолан", "Квентін Тарантіно"
   - Показується на сторінці фільму

3. **actors** (Актори)
   - Тип: Text
   - Формат: через кому
   - Приклад: "Леонардо Ді Капріо, Том Гарді, Джозеф Гордон-Левітт"

4. **duration** (Тривалість)
   - Тип: Integer (хвилини)
   - Приклад: 148 (відображається як "148 хв (2 год 28 хв)")
   - Показується як бейдж

5. **age_rating** (Віковий рейтинг)
   - Тип: String(10)
   - Можливі значення: G, PG, PG-13, R, NC-17
   - Відображається як бейдж з описом

6. **release_year** (Рік випуску)
   - Тип: Integer
   - Приклад: 2024
   - Показується як бейдж на картках

---

## 🎯 Де відображається:

### 1. **Адмін-панель** (`/admin/films`)
- Всі поля доступні для редагування
- Валідація даних (рік 1900-2030, тривалість 1-500 хв)
- Бейджи на картках фільмів (жанр, рік, тривалість)

### 2. **Список фільмів** (`/films`)
- Бейджі: жанр, рік, тривалість, віковий рейтинг
- Компактне відображення на картках

### 3. **Сторінка фільму** (`/film/<id>`)
- Інформаційна картка з усіма деталями:
  - Жанр (бейдж)
  - Рік випуску
  - Режисер
  - Тривалість (хв та години)
  - Віковий рейтинг (бейдж)
  - Актори (повний список)

### 4. **Головна сторінка** (`/`)
- Бейджі жанру та року для популярних фільмів

---

## 📝 Використання:

### Для адміністраторів:

1. Увійдіть як адміністратор
2. Перейдіть до `/admin/films`
3. При додаванні/редагуванні фільму заповніть нові поля:
   - **Жанр**: вкажіть основний жанр (наприклад: "Фантастика")
   - **Режисер**: повне ім'я режисера
   - **Актори**: перелічіть через кому
   - **Тривалість**: в хвилинах
   - **Віковий рейтинг**: оберіть зі списку
   - **Рік випуску**: рік світової прем'єри

### Для користувачів:

- Нова інформація автоматично відображається на всіх сторінках
- Зручно фільтрувати фільми за жанром, роком або тривалістю
- Бачити повну інформацію перед бронюванням

## 🎨 Дизайн:

### Бейджі:
- **Жанр** - синій (`bg-primary`)
- **Рік** - сірий (`bg-secondary`)
- **Тривалість** - блакитний (`bg-info`)
- **Віковий рейтинг** - жовтий (`bg-warning`)

### Картка інформації:
- Зручна сітка з двох колонок
- Автоматичне приховування порожніх полів
- Конвертація тривалості у години та хвилини

---

## 🔄 Міграція бази даних:

База даних була пересоздана з новими полями. Старі дані НЕ ЗБЕРЕЖЕНІ.

Якщо потрібно зберегти старі дані, використовуйте Flask-Migrate:

```bash
pip install Flask-Migrate
flask db init
flask db migrate -m "Added extended film info"
flask db upgrade
```
